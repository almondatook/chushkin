# Чушкин 1.0

Крошечная char-level языковая модель — трансформер на ≈ 0,8 млн параметров, обученный на «Евгении Онегине». Учебный проект: наглядно показать, как из чистой статистики букв прорастает псевдопушкинский текст. Модель не знает ни одного слова, но безошибочно ловит склад русской речи.

Это начинка интерактивного мини-курса и лаборатории: **https://makarina.pro/chushkin/**

## Что в репозитории

| Файл | Что делает |
|------|------------|
| `tiny_llm.py`  | обучение модели с нуля |
| `generate.py`  | генерация текста обученной моделью |
| `onegin.txt`   | корпус — «Евгений Онегин» (~142 000 символов) |
| `plot_loss.py` | график кривой обучения |

## Как запустить (Windows, для новичков)

1. Поставить Python с [python.org/downloads](https://www.python.org/downloads/) — на первом экране установщика **обязательно** галочка «Add Python to PATH».
2. Открыть PowerShell в папке проекта (в Проводнике в адресной строке набрать `powershell` и нажать Enter).
3. Поставить PyTorch (CPU-версию):
   ```
   python -m pip install torch --index-url https://download.pytorch.org/whl/cpu
   ```
4. Обучить (≈ 40 минут на обычном процессоре):
   ```
   python tiny_llm.py
   ```
5. Генерировать:
   ```
   python generate.py --temp 0.8
   python generate.py --prompt "Татьяна" --len 800
   ```

Подробный пошаговый разбор для совсем новичков — в лаборатории на makarina.pro.

## Свой текст вместо Пушкина

Замените `onegin.txt` своим корпусом (желательно ≥ 100 000 символов одного жанра, иначе модель просто зазубрит текст наизусть) и переучите.
